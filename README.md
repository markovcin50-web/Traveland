## ✨ About Project

**Trevland** — это стильный travel-лендинг, который вдохновляет пользователей на путешествия 🌴
Минималистичный дизайн, красивые локации и простая структура делают сайт удобным и современным.

---

## 🧭 Features

✨ Современный UI
🌆 Популярные направления
💸 Специальные предложения
🎯 Простая навигация
🎨 Кастомные шрифты
⚡ Быстрая загрузка

# 🌍 Trevland — Travel Landing Page

🚀 **Live Demo:**
https://markovcin50-web.github.io/Traveland/

---

## ✨ About the Project

**Trevland** — это современный лендинг для путешествий, созданный с акцентом на минимализм, эстетику и удобство пользователя.
Сайт демонстрирует популярные направления, специальные предложения и вдохновляет на новые поездки.

---

## 🧭 Features

* 🌆 Красивый travel-дизайн
* 📍 Популярные локации (Istanbul, Kuala Lumpur, Seoul)
* 💸 Специальные предложения с ценами
* 🎯 Простая навигация
* 📱 Адаптивная структура (можно доработать под mobile)
* 🎨 Использование Google Fonts

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Google Fonts (Roboto, Ribeye, Gelasio)

## 🚀 Getting Started

1. Клонируй репозиторий:

```bash
git clone https://github.com/markovcin50-web/Traveland.git
```

2. Перейди в папку проекта:

```bash
cd Traveland
```

3. Открой `index.html` в браузере
